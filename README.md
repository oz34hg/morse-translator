# Just a morse translator

## Main Features

- Translating from any latin based language into international morse code
- Translating back from international morse code into any latin based language
- An easy to use and lightweight GUI built on java swing

## Features to be implemented
- support for accents in morse code
- support for punctuation
- a better looking GUI

### Quick note

The main purpose of creating this morse translator was learning to work
with java swing
